The three languages of the web

HTML5

Here are 5 things I know about HTML:

HTML stands for HyperText Markup Language
This language is responsible for the content of a web page, WHAT you see
Some tags, like img have attributes, like src and alt
It is good practise in HTML to double space indent rather than using tabs
HTML5 is the latest of the HTML languages and introduced numerous more semantic elements

CSS

Here are 5 things I know about CSS:

  • CSS stands for Cascading Style Sheet
  • CSS is responsible for how the content is displayed on a web page
  • Within a CSS sheet, you should order the styles as they appear on a web page
  • Styling of a website should be done mobile-first. That is, developed for mobiles, and extended for larger devices using breakpoints
  • There are often many curly braces in CSS files
